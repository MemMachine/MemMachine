The MemMachine Python SDK provides a simple interface for project-scoped
memory operations.

## Install

```bash
pip install memmachine-client
```

## Hello World

```python
from memmachine import MemMachineClient


def main() -> None:
    client = MemMachineClient(base_url="http://localhost:8080")

    project = client.get_or_create_project(
        org_id="my-org",
        project_id="hello-world-project",
        description="My first MemMachine project",
    )

    memory = project.memory(
        user_id="user-alice",
        agent_id="agent-bob",
        session_id="session-1",
    )

    memory.add(content="My favorite color is blue.", role="user")
    memory.add(
        content="Got it. I will remember your favorite color is blue.",
        role="assistant",
    )

    result = memory.search("What is my favorite color?")
    print(result)


if __name__ == "__main__":
    main()
```

## Key Concepts

- `MemMachineClient`: HTTP client for MemMachine server APIs.
- `Project`: isolation boundary (`org_id` + `project_id`).
- `Memory`: context-scoped interface for `add`, `search`, `list`, and `delete`.
  Common context fields are `user_id`, `agent_id`, `session_id`, and `group_id`.
